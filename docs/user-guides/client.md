# Guide Client — Équatis

Bienvenue sur votre espace personnel pour suivre votre acquisition immobilière.

## S'inscrire

1. Rendez-vous sur https://equatis.fr/inscription
2. Renseignez vos coordonnées (prénom, nom, email, téléphone, adresse postale)
3. Choisissez un mot de passe (8 caractères, 1 majuscule, 1 chiffre)
4. Validez les conditions d'utilisation
5. Vous recevez un **email de confirmation** sous quelques minutes — cliquez sur le lien (valable 24h)

Une fois confirmé, votre compte est en attente d'**association à un dossier**. Le collaborateur Équatis qui suit votre acquisition vous associera dans les jours qui suivent.

## Tableau de bord

Une fois associé, vous voyez :
- **Votre programme** et le **lot** acquis (référence, surface, type, prix TTC)
- **Le statut actuel** de votre dossier
- **Une barre de progression** sur les 8 étapes :
  1. Lead reçu
  2. Réservation envoyée
  3. Signature en attente
  4. Chez le notaire
  5. Offre de prêt reçue
  6. Acte signé
- **Votre prochaine étape**
- **Le contact de votre collaborateur référent** (clic pour message direct)

## Mes documents

### Pièces à déposer
Votre collaborateur configure les documents à déposer. Pour chaque pièce :
- Glissez-déposez le fichier ou cliquez sur "parcourez"
- Formats acceptés : **PDF, JPG, PNG, DOCX** — taille max **20 Mo**
- Le fichier est analysé par antivirus avant d'être marqué **Sécurisé** (généralement < 1 min)

Vous pouvez **remplacer** une pièce déjà déposée à tout moment.

### Documents reçus
Documents partagés par votre collaborateur (plans, notices, contrat de réservation, etc.). Téléchargement via lien signé 15 min — uniquement après scan antivirus validé.

## Messagerie

Conversation directe avec votre collaborateur référent. Notification email à chaque nouveau message.

Astuce clavier : **Ctrl+Entrée** (Windows/Linux) ou **⌘+Entrée** (Mac) pour envoyer.

## Mon profil

### Coordonnées
Modifier prénom, nom, téléphone, adresse. Vos coordonnées sont **chiffrées** en base (AES-256).

### Mot de passe
Changement nécessite l'ancien mot de passe. Toutes vos autres sessions sont déconnectées automatiquement.

### Suppression du compte (RGPD)
Cliquez sur **Demander la suppression de mon compte**. Votre demande est transmise à l'administrateur Équatis et traitée sous 30 jours conformément au RGPD.

Pendant le traitement, votre compte est suspendu (vous ne pouvez plus vous connecter).

## Sécurité

- **Vos données personnelles sont chiffrées** en base (téléphone, adresse)
- **HTTPS partout** : aucune donnée ne transite en clair
- **Déconnexion automatique** après 30 minutes d'inactivité
- **Notifications email** : vous pouvez configurer les types de notifications via votre profil (Phase 5+)
- **Aucun mot de passe en clair** : nous ne voyons jamais votre mot de passe

## En cas de souci

| Problème | Solution |
|---|---|
| Mot de passe oublié | https://equatis.fr/mot-de-passe-oublie |
| Email de confirmation non reçu | Vérifier les spams. Sinon, contactez votre collaborateur. |
| Upload échoue | Vérifier taille (< 20 Mo) et format (PDF/JPG/PNG/DOCX) |
| Document refusé après scan | Le fichier contient un virus. Réessayer avec un fichier sain. |
