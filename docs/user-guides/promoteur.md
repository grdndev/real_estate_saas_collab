# Guide Promoteur — Équatis

## Accéder à votre espace

1. Recevez l'email d'invitation envoyé par l'admin Équatis
2. Cliquez sur **Définir mon mot de passe** (lien valable 7 jours)
3. Choisissez un mot de passe (8 caractères, 1 majuscule, 1 chiffre minimum)
4. Connectez-vous sur https://equatis.fr/connexion

Vous arrivez sur la liste de **vos programmes** — uniquement ceux auxquels l'admin vous a assigné.

## Tableau de bord d'un programme

3 indicateurs majeurs :
- **Lots vendus** / total + barre de progression
- **Lots réservés** / total
- **Chiffre d'affaires réalisé** sur lots SOLD vs objectif (configuré par l'admin)

Statistiques détaillées : disponibles, réservés, vendus, prix moyen au m².

## Grille de prix & lots (CDC §5.4)

Vue lecture seule de tous les lots du programme :
- Référence, surface, étage, type
- Prix HT / TVA / Prix TTC
- Statut (disponible, réservé, vendu, retiré)

Un bouton **Exporter (CSV)** permet de récupérer la grille pour usage externe (Excel, comptabilité, présentation).

## Trésorerie prévisionnelle (CDC §5.6)

Saisie mensuelle sur les **12 mois glissants** à partir du mois courant :
- **Entrées prévisionnelles** (ventes attendues, etc.)
- **Dépenses** (chantier, frais de commercialisation, etc.)
- **Solde** calculé automatiquement (vert si positif, rouge sinon)

Le bouton **Enregistrer** ne s'active que pour les lignes modifiées.

3 cards de synthèse en haut : total entrées, total dépenses, solde prévisionnel.

Export CSV disponible.

## Suivi des ventes (CDC §5.7)

Vue **lecture seule sans données personnelles** :
- Liste des dossiers du programme
- Référence, lot, statut, dates
- Statistiques : taux de conversion lead → acte, durée moyenne lead → acte

Les noms des acquéreurs ne sont **jamais visibles** dans cette vue (cloisonnement RGPD).
