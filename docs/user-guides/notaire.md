# Guide Notaire — Équatis

## Accéder à votre espace

1. Recevez l'email d'invitation envoyé par l'admin Équatis
2. Cliquez sur **Définir mon mot de passe** (lien valable 7 jours)
3. Choisissez un mot de passe (8 caractères, 1 majuscule, 1 chiffre minimum)
4. Connectez-vous sur https://equatis.fr/connexion

## Dossiers reçus

À chaque transmission par un collaborateur, vous recevez un email + une notification.

La liste affiche tous les dossiers qui vous sont **explicitement transmis** — vous ne voyez jamais d'autres dossiers (cloisonnement strict CDC §6.1).

Statuts visibles :
- **Reçu** — en attente de prise en charge
- **En cours** — en cours de traitement
- **Acte prêt à signer** — finalisé en attente du rendez-vous
- **Acte signé** — clôturé

## Fiche dossier

### Informations
Référence, programme, lot, surface, prix, dates de transmission, identité de l'acquéreur.

### Documents transmis
Téléchargement individuel de chaque document via URL signée 15 min (CDC §11.2). Les documents au statut **Sécurisé** (vert) sont scannés antivirus et téléchargeables. Les autres restent indisponibles tant que le scan n'a pas validé.

### Pièces demandées
Liste des pièces que le collaborateur a demandées au client + statut (déposée / en attente).

### Timeline
Toutes les étapes du dossier (création, transmission, événements signalés).

## Actions disponibles

### Marquer l'acte comme signé
Action irréversible (modale de confirmation) :
- Le dossier passe en `Acte signé`
- Le lot passe en `Vendu`
- Le client + les collaborateurs reçoivent un email + une notification

### Bloquer le dossier
Marque le dossier comme bloqué (statut visible par tous). Ajoutez un commentaire via le signalement de pièce manquante pour expliquer.

### Signaler une pièce manquante
Saisissez le libellé de la pièce manquante. Une `Pièce demandée` est automatiquement créée côté collaborateur, taggée `[Demandé par notaire]`. Le collaborateur reçoit une notification.

## Confidentialité

Vous ne voyez **que** les dossiers qui vous ont été transmis. Aucun moyen d'accéder à un dossier transmis à un autre notaire, même via manipulation d'URL (vérification serveur systématique).

Toutes vos actions sont tracées dans le journal d'audit (5 ans, CDC §8.6).
